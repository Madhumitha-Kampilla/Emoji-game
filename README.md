# Emoji Game 🎮

A fun React-based game where users select emojis without repeating.

## 🚀 Features
- Random emoji selection
- Score tracking system
- Game over and restart functionality
- Responsive UI

## 🛠 Tech Stack
- React.js
- JavaScript
- CSS

## ▶️ Run Locally
```bash
npm install
npm start
```
